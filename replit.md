# Vibe Coding Baseline

## Overview
A baseline project for "From Isolated Prompts to Shared Context: Making Vibe Coding Production-Ready in Enterprise Teams". Currently serves a simple landing page using Node.js and Express.

## Project Architecture
- **Runtime**: Node.js 20
- **Framework**: Express
- **Port**: 5000 (bound to 0.0.0.0)
- **Entry point**: `index.js`
- **Static files**: `public/` directory

## File Structure
```
index.js          - Express server entry point
public/
  index.html      - Landing page
  styles.css      - Stylesheet
package.json      - Node.js dependencies
```

## Running
The application starts via `node index.js` and serves on port 5000.
